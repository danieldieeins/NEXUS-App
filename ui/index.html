2024.2-beta.1 ▪ Argrium

Published on 29th January 2024 2:44am

Info: With this version of the application we restart the versioning. The old public beta phase is over, and we are now preparing for the full version. The plan is for the first full version to come in February 2024. That's why this is version 2024.2-beta.1 - as it is the first beta for the upcoming version in month 2.

Added:
+ Added window resizing functionality.
+ Added official instances to instance adder.
+ Added a changelog tab to News & updates.
+ Some buttons will start an animation on hover.
+ New dark mode theme.
+ New light mode theme.

Changed:
• The entire user interface has been rewritten from scratch.
• The appearance of the pop-up windows has been redesigned.
• Renamed Start/News to News & updates.
• Changed the default zyneon theme.
• Official instances will now show up in the instances list.
• Not installed official instances will no longer show up in the instances list.

Removed:
- HTTPServer (Webserver)
- "Official" section from instances
- "Resources" menu entry
- "Wiki" menu entry
- "Zyneon Drive" menu entry
- Old news from open beta
